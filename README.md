##  About Me  
Hi! I'm **Fujii**, a creative developer blending **web development**, **3D art**,  
**game creation**, and **cinematic editing** into a single artistic identity.

I love building expressive digital experiences — from modern UIs to stylized 3D,  
from engaging videos to atmospheric game worlds.
